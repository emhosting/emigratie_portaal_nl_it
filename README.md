# Emigratie Portaal Plugin

Deze plugin biedt een portaal voor het beheren van taken en categorieën.

## Installatie

1. Upload de pluginbestanden naar de `/wp-content/plugins/emigratie-portaal` directory, of installeer de plugin via het WordPress plugin scherm.
2. Activeer de plugin via het 'Plugins' scherm in WordPress.
3. Gebruik de shortcode `[ep_user_interface]` om de gebruikersinterface weer te geven.

## Functies

- Beheer van taken en categorieën
- AJAX-ondersteuning voor dynamische updates
- Admin interface voor het beheren van categorieën en taken
